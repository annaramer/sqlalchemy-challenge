Available Routes:

/api/v1.0/precipitation
- List of prior (or 1) year rain totals from all stations

/api/v1.0/stations
- List of Station numbers and names

/api/v1.0/tobs
- List of prior (or 1) year temperatures from Station USC00519281

/api/v1.0/start
- When given the start date (YYYY-MM-DD), calculates AVG/MAX/MIN temperature for all dates greater than and equal to the start date

/api/v1.0/start/end
- When given the start and the end date (YYYY-MM-DD), calculates AVG/MAX/MIN temperature for dates between the start and end date inclusive
